Explanation of the dataset

It is crucial to understand the data foundation of the network, prior to setting up the network. As already mentioned the data for this study is based on the wikipidedia lists of important artists wihtin each of the five genres; Pop, Hiphop, RnB, Rock, Country. The data has been downloaded by reading the list of artists for each genre using the wikipedia API. The result of the API is a html file containing the list of artists of the given genre. By using regular expression on the html files it is possible to extract and save the name of each artist into a pandas DataFrame. Hence this process results in a dataframe containing the list of all artists with the attributes name, genre, name_ and link_name. Then it is possible to use the same wikipedia API to download the wikipedia page of every artist. The downloaded wikipedia pages is then saved as txt files with the name of then given artist.
